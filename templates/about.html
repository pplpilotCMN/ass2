Project Overview

This project is a Flask-based web application that visualises sentiment analysis of U.S. airline tweets. It processes Twitter data and presents it through interactive visualisations and a sortable table.

How It Works

  • Flask Backend: Handles routing and serves different pages (about, advanced, basic, creative, and index.)
  • Data Processing: Reads airline sentiment data from a CSV file (Kaggle_TwitterUSAirlineSentiment.csv) .
  • Bubble Chart: Uses D3.js to visualise sentiment distribution per airline.
  • Sortable Table: Displays all data from the CSV file with sorting functionality.
  • Sentiment Table: Displays 40 rows and limited columns with `airline_sentiment_confidence` in ascending order.

Key Features

  • Interactive Bubble Chart, sentiment visualisation with D3 force-directed bubbles.
  • An all columns Sortable table.
  • CSV data handling with error checking.
  • Dynamic filtering of airline sentiment data.

Technologies Used

  • Flask (Python backend)
  • D3.js (Data visualisation)
  • HTML, CSS, JavaScript (Frontend)
  • CSV Data Processing
  • unittest for testing
  • prettytable for clear test results
